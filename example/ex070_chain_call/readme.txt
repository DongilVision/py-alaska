================================================================================
              Chain Call Example - 중첩 RMI 호출 & Signal 이벤트 테스트
================================================================================

목적: 중첩 호출(Chain RMI)과 이벤트(Signal)를 시험

- 토큰 10개 보유
- source -> mid -> dest 순으로 중첩 RMI 호출
- dest에서 signal 이벤트로 source에 토큰 반환
- 라운드 시간 측정 (전달 → signal 수신)


================================================================================
                              시나리오 흐름도
================================================================================

    ┌─────────────┐        ┌─────────────┐        ┌─────────────┐
    │   SOURCE    │        │     MID     │        │    DEST     │
    │ (토큰 보유) │        │ (중첩 호출) │        │(이벤트 발신)│
    └──────┬──────┘        └──────┬──────┘        └──────┬──────┘
           │                      │                      │
           │  RMI: relay_token()  │                      │
    [1]    │─────────────────────>│                      │
           │  (토큰 제거)         │                      │
           │                      │  RMI: consume_token()│
    [2]    │                      │─────────────────────>│  (중첩 호출)
           │                      │                      │
           │                      │                      │ [3] 토큰 처리
           │                      │                      │
           │    Signal: token.returned                   │
    [4]    │<─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─│  (이벤트)
           │                      │                      │
           │ [5] 토큰 다시 채움   │                      │
           │     라운드 시간 계산 │                      │


================================================================================
                              핵심 기능
================================================================================

1. 중첩 RMI 호출 (Chain Call)
   - source.mid.relay_token(token)
   - mid 내부에서 dest.consume_token(token) 호출
   - 호출 체인: source → mid → dest

2. Signal 이벤트 (비동기 브로드캐스트)
   - dest에서 self.signal.token.returned.emit(token)
   - source의 on_token_returned(signal) 자동 호출
   - on_ 패턴으로 자동 구독

3. 라운드 시간 측정
   - 토큰에 send_time 포함하여 전달
   - signal 수신 시 라운드 시간 계산


================================================================================
                              파일 구조
================================================================================

example/chain_call/
├── config.json         # Task 설정
├── main.py             # 실행 진입점
├── task_source.py      # 토큰 보유, 중첩 RMI 호출, 라운드 시간 측정
├── task_mid.py         # RMI 중계 (중첩 호출)
└── task_dest.py        # 토큰 처리, Signal 이벤트 발신


================================================================================
                              핵심 코드 패턴
================================================================================

1. 중첩 RMI 호출
   ─────────────────────────────────────────────────────────────────────────────
   # source에서 mid 호출
   token = {"id": "TOKEN-00", "send_time": time.time()}
   result = self.mid.relay_token(token)

   # mid 내부에서 dest 호출 (중첩)
   def relay_token(self, token):
       return self.dest.consume_token(token)

2. Signal 이벤트 발신
   ─────────────────────────────────────────────────────────────────────────────
   self.signal.token.returned.emit(token)

3. on_ 패턴 자동 구독
   ─────────────────────────────────────────────────────────────────────────────
   def on_token_returned(self, signal):
       token = signal.data
       round_time = (time.time() - token["send_time"]) * 1000

                              실행 방법

python -m example.chain_call.main
